1. FinanceAI – Smart Bank Statement Analyzer

FinanceAI is a financial analytics web application that transforms raw bank transaction data into meaningful insights using NLP and statistical analysis.

2. Features

- Upload bank statement CSV files
- NLP-based transaction categorization
- Anomaly detection using statistical methods (2σ rule)
- Monthly expense trend analysis
- Category-wise spending visualization
- Financial health score calculation
- Smart financial suggestions

3. Tech Stack

- Python
- Pandas
- Streamlit
- Matplotlib
- NLP (Keyword-based classification)

4. How It Works

- Upload CSV file
- Data cleaning & preprocessing
- Transaction categorization using NLP
- Anomaly detection using mean & standard deviation
- Visualization of insights via dashboard
